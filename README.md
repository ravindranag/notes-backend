# Notes REST API

# Instructions 

1. Install dependencies
   ```
   npm i
   ```

2. Copy .env.example > .env.development
   ```
   cp .env.example .env.development
   ```

3. Enter values of secrets in .env.development
4. Run migrations
   ```
   npm run db:migrate
   ```
5. Start app in development mode
   ```
   npm run start:dev
   ```
